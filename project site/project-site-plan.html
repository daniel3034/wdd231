Project Site Plan

1. Site Name

Online Recipe Repository

Reason for selection: This name clearly conveys the site's purpose of being a centralized location for various recipes, making it easy for users to understand what the site offers.

2. Site Purpose

The site provides a platform for users to browse, search, and submit recipes. It will offer features like recipe categories, user-generated content, and a rating system to enhance user engagement.

3. Scenarios

Scenario 1: How can I find healthy recipes for dinner?

Scenario 2: Can I submit my own recipes to share with others?

4. Color Schema

Color 1: Light Green (#c8e6c9) - Used for background and section highlights.

Color 2: Dark Green (#2e7d32) - Used for headings and buttons.

5. Typography

Font 1: 'Roboto', sans-serif - Used for body text.

Font 2: 'Lobster', cursive - Used for headings.

Font 3 (optional): 'Open Sans', sans-serif - Used for buttons and links.

6. Wireframe

Design a wireframe for your homepage layout:

Mobile View:

Header with logo and navigation.

Main section with featured recipes.

Search bar.

Footer with links to social media.

Desktop View:

Larger header with logo and navigation.

Side menu with categories.

Main content area for featured recipes and search results.

Footer with links and contact information.
